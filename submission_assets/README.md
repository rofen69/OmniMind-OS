# OmniMind OS - Submission Assets Directory

This folder is organized to separate submission assets (video, screenshots, presentations, writeups) from the core Python source code of OmniMind OS.

---

## Folder Structure

```
submission_assets/
├── README.md               # This directory guide
├── video/                  # Unlisted YouTube link and script assets
│   └── YouTube_Link.txt    # Direct Link to 2-3 minute demo video
├── screenshots/            # Raw image captures for Kaggle writeup
│   ├── 01_CLI_Startup.png
│   ├── 02_Discovery_Table.png
│   ├── 03_Timeline_Output.png
│   └── 04_Web_UI_Dashboard.png
└── presentation/           # Presentation slides (PDF/PPTX)
    └── Slide_Deck.pdf
```

---

## Core Assets Checklist

1. **Video Demo:** Located in `video/YouTube_Link.txt`. Covers the Zero-Token multi-agent execution in the terminal CLI and the Flask Web UI.
2. **Kaggle Writeup Assets:** Images to be embedded directly into the Kaggle discussion forum post are organized in `screenshots/`.
3. **Architecture Slide:** Embedded within `presentation/Slide_Deck.pdf` (derived from the Mermaid charts in `docs/02_SYSTEM_ARCHITECTURE.md`).
