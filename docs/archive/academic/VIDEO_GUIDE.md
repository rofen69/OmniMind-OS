# OmniMind OS - Video Recording Guide

This step-by-step guide outlines how to record and structure the **5-minute demo video** for the Kaggle AI Agent Capstone submission.

---

## Video Outline & Timing (Total: 5:00)

| Time | Scene Name | On-Screen Action | Script / Speech Guide |
|------|------------|------------------|----------------------|
| **0:00 - 0:45** | **The Hook & The Problem** | Facecam or Title Slide + Mermaid Diagram (`docs/02_SYSTEM_ARCHITECTURE.md`) | Explain the **Chatbot Trap**. Explain why a modular Operating System model beats brittle monolithic prompt wrapping. |
| **0:45 - 2:00** | **Terminal CLI & Heuristic Planner** | Run `python -m src.main` and choose option `2` | Input: `"Research Quantum Computing and format it as a document then save the file"`. Highlight the dynamic 3-step execution plan generated by the Heuristic Planner. |
| **2:00 - 3:00** | **Discovery & Selection** | CLI output scrolling (Table Renderer) | Point out the candidate agents table. Explain how the selector evaluates cost, score, and licensing based on user preferences. |
| **3:00 - 4:00** | **State Execution & Memory** | VS Code side-by-side with CLI | Show `src/memory/session.py` and explain how agents communicate via shared global keys instead of calling each other. Show the newly created file in the `workspace/` folder. |
| **4:00 - 4:40** | **System Command Layer** | Input commands: `help`, `agents`, `status` in the CLI | Type administrative commands to demonstrate the deterministic System Command Layer running before the planner. |
| **4:40 - 5:00** | **Closing & Future Roadmap** | Slide Deck or README checklist | Reiterate the key differentiator: **Zero-Token, Zero-Auth deployability**. Mention MCP and Google ADK as Version 2 targets. |

---

## Recommended Scenes & Captures

1. **Scene 1: Architecture Mermaid Rendering**
   - *Command:* Open `docs/02_SYSTEM_ARCHITECTURE.md` in a preview pane.
   - *Focus:* Show the unidirectional flow: User Input -> Planner -> Discovery Pipeline -> Executor -> Timeline.
2. **Scene 2: Booting the CLI**
   - *Command:* `python -m src.main`
   - *Focus:* Show options `1` and `2` cleanly. Enter `2`.
3. **Scene 3: Heuristic Verb Mapping**
   - *Prompt:* `"Research solar energy in Bangladesh and write it to a file"`
   - *Focus:* Scroll up to show the planner mapping "Research" to `research` and "write" to `file` capabilities.
4. **Scene 4: CLI Candidate Table**
   - *Focus:* Highlight the dynamic columns (`Installed`, `Score`, `License`) to show the judge how selection tiebreakers are rendered.

---

## Animations and Visual Help

- Use **mouse pointer highlights** to draw attention to the specific Agent Selection green checkmarks (`✅`) in the timeline.
- Zoom in on the terminal during command outputs to ensure the text is highly readable on small screens.
- Do a **live file expansion animation** in VS Code showing `workspace/` folders populating in real-time as the CLI finishes step 3.
